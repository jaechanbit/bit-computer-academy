# 비트컴퓨터학원 홈페이지 화면설계서

## 1. 문서 개요

- 대상 서비스: 비트컴퓨터학원 홈페이지
- 운영 도메인: bit-edu.com
- 목적:
  - 방문자가 교육과정을 검색하고 상담 문의를 남길 수 있게 한다.
  - 관리자가 교육과정을 등록/수정하고 문의 내역을 관리할 수 있게 한다.
- 기준 문서: `docs/requirements.md`

## 2. 전체 화면 목록

## 4. 방문자 화면 설계

## V-01. 홈

### 목적

방문자가 비트컴퓨터학원의 핵심 정보를 빠르게 이해하고, 원하는 교육과정을 검색하거나 문의로 이동하게 한다.

### 주요 섹션

1. 히어로 영역
2. 과정 검색 영역
3. 과정 분류 바로가기
4. 추천/모집중 과정
5. 학원 강점
6. 상담 문의 유도
7. 위치 요약

### 화면 구성

```text
[헤더]

[히어로]
비트컴퓨터학원
남원에서 배우는 실무 중심 IT 교육
일반과정부터 국민내일배움카드과정까지, 원하는 교육과정을 쉽게 찾아보세요.
[교육과정 보기] [상담 문의]

[과정 검색]
검색어 입력: 과정명을 입력하세요
분류 선택: 전체 / 일반과정 / 국민내일배움카드과정
[검색]

[과정 분류]
[일반과정] [국민내일배움카드과정]

[모집중 과정]
과정 카드 3~6개

[학원 강점]
실무 교육 / 자격증 대비 / 지역 밀착 상담

[오시는 길 요약]
주소, 전화번호, 지도 일부 또는 버튼

[푸터]
```

### 표시 데이터

- 학원명
- 대표 문구
- 과정 검색 입력값
- 과정 분류
- 모집중 과정 목록
- 주소
- 전화번호

### 주요 버튼

| 버튼 | 동작 |
| --- | --- |
| 교육과정 보기 | `/courses`로 이동 |
| 상담 문의 | `/inquiry`로 이동 |
| 검색 | 검색 조건을 포함해 `/courses`로 이동 |
| 과정 카드 | 해당 과정 상세로 이동 |

- 방문 상담 및 전화 상담 가능

[운영 과정 요약]
일반과정 / 국민내일배움카드과정

## V-03. 교육과정 목록

### 목적

방문자가 과정명을 검색하거나 분류별로 교육과정을 찾아볼 수 있게 한다.

### 화면 구성

```text
[헤더]

[페이지 제목]
교육과정

[검색/필터 영역]
검색어 입력
분류: 전체 / 일반과정 / 국민내일배움카드과정
모집상태: 전체 / 모집중 / 예정 / 마감
[초기화] [검색]

[과정 목록]
카드 또는 리스트 반복
- 과정명
- 분류
- 짧은 소개
- 개강일
- 교육 기간
- 교육 시간
- 국민내일배움카드 적용 여부
- 모집 상태
[자세히 보기] [문의하기]

[푸터]
```

### 필터 항목

| 항목 | 타입 | 옵션 |
| --- | --- | --- |
| 검색어 | 텍스트 | 과정명, 소개 검색 |
| 과정 분류 | 선택 | 전체, 일반과정, 국민내일배움카드과정 |
| 모집 상태 | 선택 | 전체, 모집중, 예정, 마감 |

### 초기 과정 목록

일반과정:
- ITQ
- 정보처리
- 사무자동화
- 포토샵
- 일러스트
- 컴퓨터그래픽스
- 엑셀실무
- 컴퓨터활용능력1급
- 워드프로세서

국민내일배움카드과정:
- ITQ마스터
- 컴퓨터활용능력1급실기
- ITQ한글/엑셀
- 컴퓨터활용능력2급+워드프로세서

### 주요 버튼

| 버튼 | 동작 |
| --- | --- |
| 검색 | 조건에 맞는 과정 목록 표시 |
| 초기화 | 필터 초기화 |
| 자세히 보기 | 과정 상세로 이동 |
| 문의하기 | 선택 과정이 입력된 문의 화면으로 이동 |

## V-05. 문의/신청

### 목적

방문자가 관심 과정에 대해 단순 상담 문의를 접수한다.

### 화면 구성

```text
[헤더]

[페이지 제목]
상담 문의

[안내 문구]
문의 내용을 남겨주시면 확인 후 연락드리겠습니다.

[문의 폼]
이름 *
연락처 *
이메일
관심 과정
국민내일배움카드 보유 여부
희망 상담 시간
문의 내용 *
개인정보 수집 및 이용 동의 *
[문의 접수]

[전화 상담 안내]
대표전화 063-626-9060

[푸터]
```

### 입력 항목

| 항목 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| 이름 | 텍스트 | 예 | 신청자 이름 |
| 연락처 | 전화번호 | 예 | 연락 가능한 번호 |
| 이메일 | 이메일 | 아니오 | 선택 입력 |
| 관심 과정 | 선택 | 아니오 | 과정 상세에서 진입 시 자동 선택 |
| 국민내일배움카드 보유 여부 | 선택 | 아니오 | 예 / 아니오 / 잘 모르겠음 |
| 희망 상담 시간 | 텍스트 또는 선택 | 아니오 | 예: 오전, 오후, 저녁 |
| 문의 내용 | 긴 텍스트 | 예 | 상담 내용 |
| 개인정보 동의 | 체크박스 | 예 | 동의해야 제출 가능 |

### 유효성 검사

- 이름은 필수
- 연락처는 필수
- 문의 내용은 필수
- 개인정보 수집 및 이용 동의 필수
- 필수값 누락 시 항목별 오류 메시지 표시

### 제출 후 처리

- 문의 데이터 저장
- 완료 메시지 표시
- "문의가 접수되었습니다. 확인 후 연락드리겠습니다." 문구 노출

## 7. 반응형 설계 기준

### PC

- 헤더 메뉴 전체 노출
- 과정 목록은 2~3열 카드 또는 테이블형 레이아웃
- 관리자 화면은 좌측 사이드바 사용 가능

### 모바일

- 헤더 메뉴는 메뉴 버튼으로 접기
- 과정 목록은 1열 카드
- 문의 버튼과 전화 버튼을 쉽게 누를 수 있게 배치
- 입력 폼은 한 줄에 하나의 항목만 표시
