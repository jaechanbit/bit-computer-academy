# 비트컴퓨터학원 홈페이지 요구사항 정의서

## 1. 프로젝트 개요

- 프로젝트명: 비트컴퓨터학원 홈페이지
- 업종: 교육서비스 / 학원
- 운영 도메인: bit-edu.com
- 주요 목적:
  - 학원에서 제공하는 교육정보를 방문자에게 알린다.
  - 방문자가 원하는 교육과정을 직접 검색하고 상세 정보를 확인할 수 있게 한다.
  - 방문자가 관심 있는 교육과정에 대해 문의 또는 신청을 남길 수 있게 한다.
  - 관리자가 교육과정을 등록, 수정, 삭제, 공개 상태 관리할 수 있게 한다.

## 2. 서비스 범위

### 2.1 방문자용 서비스

방문자는 별도 로그인 없이 교육과정을 탐색하고 문의 또는 신청을 할 수 있다.

주요 기능:
- 학원 소개 확인
- 교육과정 목록 확인
- 교육과정 검색 및 필터링
- 교육과정 상세 정보 확인
- 교육과정 문의 신청
- 일반 문의 신청

### 2.2 관리자용 서비스

관리자는 관리자 페이지에서 교육과정과 신청 내역을 관리할 수 있다.

주요 기능:
- 관리자 로그인
- 교육과정 등록
- 교육과정 수정
- 교육과정 삭제 또는 비공개 처리
- 과정 카테고리 관리
- 문의/신청 내역 확인
- 문의 처리 상태 변경

## 3. 사용자 역할

| 역할 | 설명 | 주요 행동 |
| --- | --- | --- |
| 방문자 | 교육과정 정보를 찾는 일반 사용자 | 과정 검색, 상세 확인, 문의 신청 |
| 수강 희망자 | 특정 과정에 관심이 있는 방문자 | 과정 상세 확인, 신청서 제출 |
| 관리자 | 학원 운영 담당자 | 과정 등록/수정/관리, 문의 확인 |

## 4. 교육과정 분류

초기 교육과정은 다음 두 분류를 기본으로 한다.

1. 일반과정
   - 학원에서 자체 운영하는 일반 교육과정
   - 초기 과정:
     - ITQ
     - 정보처리
     - 사무자동화
     - 포토샵
     - 일러스트
     - 컴퓨터그래픽스
     - 엑셀실무
     - 컴퓨터활용능력1급
     - 워드프로세서

2. 국민내일배움카드과정
   - 국민내일배움카드로 수강 가능한 교육과정
   - 과정별 지원 여부, 훈련 기간, 자부담금 등 안내 필요
   - 초기 과정:
     - ITQ마스터
     - 컴퓨터활용능력1급실기
     - ITQ한글/엑셀
     - 컴퓨터활용능력2급+워드프로세서

추후 확장 가능 분류:
- 국비지원과정
- 재직자과정
- 취업연계과정
- 단기특강
- 자격증과정

### 5.4 문의/신청

목적:
- 방문자가 관심 과정에 대해 쉽게 상담 문의를 남길 수 있게 한다.
- 온라인 신청은 정식 수강 신청이 아니라 단순 문의 접수로 처리한다.

필수 입력 항목:
- 이름
- 연락처
- 관심 과정
- 문의 내용
- 개인정보 수집 및 이용 동의

선택 입력 항목:
- 이메일
- 희망 상담 시간
- 국민내일배움카드 보유 여부

처리 결과:
- 제출 완료 메시지 표시
- 관리자 페이지에 신청 내역 저장
- 관리자가 전화 또는 추후 연결될 카카오톡 상담을 통해 후속 안내

필수 표시 항목:
- 학원명
- 주소: 전북특별자치도 남원시 의총로 81-10
- 대표 연락처: 063-626-9060
- 지도
- 길찾기 링크

지도 구현 방향:

- 추후 운영 환경에서 지도 API 키가 필요하면 별도 발급 후 적용한다.

### 5.6 카카오톡 상담

초기 정책:
- 카카오톡 상담은 추후 연결한다.
- 홈페이지에는 "카카오톡 상담 준비중" 또는 "전화 상담"을 우선 노출한다.

추후 연결 방법:
- 카카오톡 채널을 개설한다.
- 카카오톡 채널 관리자센터에서 채널 URL 또는 상담 버튼 링크를 발급한다.
- 홈페이지의 상담 버튼에 해당 URL을 연결한다.
- 필요 시 카카오톡 채널 추가 버튼 또는 채팅 상담 버튼을 노출한다.

## 6. 관리자 페이지 요구사항

필수 기능:

- 비밀번호는 평문 저장 금지
- 비밀번호 해시 저장

## 8. 비기능 요구사항

- 모바일과 PC에서 모두 보기 편한 반응형 화면이어야 한다.
- 방문자가 교육과정을 빠르게 찾을 수 있도록 검색과 필터를 제공한다.
- 관리자 기능은 인증된 사용자만 접근할 수 있어야 한다.
- 문의 신청 시 개인정보 수집 및 이용 동의가 필요하다.
- 추후 과정 분류와 상세 항목이 늘어날 수 있도록 확장 가능한 구조로 설계한다.
- 로고는 추후 제공되는 파일을 반영한다.
- 브랜드 색상은 최신 교육/IT 서비스 느낌에 맞춰 신뢰감 있는 모던 톤으로 제안한다.

추천 브랜드 색상 방향:
- Primary: 딥 블루 또는 인디고 계열
- Accent: 민트, 시안, 라임 중 하나
- Neutral: 화이트, 쿨 그레이, 차콜
- 사용 의도: IT 교육기관의 전문성과 학습 서비스의 친근함을 함께 전달

## 8.1 데이터 저장 방식

데이터 저장 방식은 데이터베이스를 사용한다.

초기 추천:
- PostgreSQL

추천 이유:
- 교육과정, 문의, 관리자 계정처럼 관계형 데이터 관리에 적합하다.
- 검색, 필터, 상태 관리, 관리자 기능 확장에 안정적이다.
- 대부분의 호스팅/VPS/클라우드 환경에서 지원이 좋다.

대안:
- MySQL 또는 MariaDB
- 호스팅 업체가 PostgreSQL을 지원하지 않는 경우 MySQL/MariaDB를 사용한다.

## 8.2 배포 방식

배포 방식은 호스팅을 사용한다.

권장 방향:
- 프론트엔드와 백엔드를 함께 운영할 수 있는 웹 호스팅 또는 VPS 호스팅
- 데이터베이스 제공 여부 확인 필요
- SSL 인증서 적용 필수
- bit-edu.com 도메인 연결

확인 필요:
- 현재 확보한 도메인의 DNS 관리 업체
- 사용할 호스팅 업체
- 호스팅에서 Node.js, PHP, Python 중 어떤 서버 환경을 지원하는지
- 데이터베이스 종류와 용량
